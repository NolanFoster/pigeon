Send a message

curl -d "Hello!" https://your-worker.dev/mytopic

With title and priority:

curl -H "X-Title: Alert" -H "X-Priority: 5" \
     -d "Server is down!" https://your-worker.dev/mytopic

As a todo (renders with a checkbox; check it in the UI to mark complete):

curl -H "X-Tags: todo" -d "Buy milk" https://your-worker.dev/groceries
